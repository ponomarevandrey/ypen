# How to use

Inline:

```pug
span(class=`text_display_inline`) Lorem Ipsum is simply dummy text of the printing and typesetting industry
```

Block:

```pug
p(class=`text text_display_block text_size_xl`) Lorem Ipsum is simply dummy text of the printing and typesetting industry. Lorem Ipsum has been the industry's standard dummy text ever since the #[span(class=`text_display_inline`) 1500s], when an unknown printer took a galley of type and scrambled it to make a type specimen book.
```
